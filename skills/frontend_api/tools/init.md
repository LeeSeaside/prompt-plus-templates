# Frontend API - 初始化提示词

## 使用方法
复制以下内容发送给 AI，初始化前端 API 配置。

---

```
请执行 frontend_api 技能初始化：

1. 扫描前端项目结构
2. 识别技术栈：框架、HTTP 客户端、TypeScript
3. 分析 API 规范：
   - API 文件位置和命名规范
   - 类型定义位置和命名规范
   - 请求封装方式（axios 实例配置）
   - 导出方式（命名导出/默认导出）
4. 提取通用类型定义（Result、PageResult 等）
5. 将结果写入 `.ai-workspace/skills/frontend_api/context.md`

完成后回复："✅ frontend_api 初始化完成"
```
